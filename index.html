User Manual

Introduction

Welcome to the AIDevX AF, AIDevX UI and AIDevX AI User Guide, your essential resource for understanding and utilizing the tools in the AIDevX ecosystem. This guide helps users efficiently manage roles, users, menus, APIs, and user interfaces with AIDevX AF, as well as handle projects, assistants, and workflows using AIDevX AI. Designed for both beginners and experienced administrators, this guide includes clear instructions, practical advice, and visual examples to support your tasks.

About AIDevX AF

AIDevX AF (Application Framework) is a powerful tool designed for building and customizing the backend of applications. With an intuitive interface, AIDevX AF enables administrators to:

  • Define and assign user roles.
  • Manage user accounts and permissions.
  • Organize menus and navigation structures.
  • Create, edit, and manage APIs.
  • Design user interfaces with pre-built components.
  • Test and refine code in an integrated playground.

These features allow organizations to create scalable and robust systems tailored to their needs.

About AIDevX AI

AIDevX AI extends AIDevX AF’s capabilities with advanced artificial intelligence tools, supporting developers, project managers, and data scientists. Key features include:

  • Managing and tracking projects to improve workflows.
  • Integrating AI assistants for specific tasks.
  • Conducting guided, interactive problem-solving chats.
  • Using prompt templates to streamline task execution.
  • Uploading and editing documents or images efficiently.

AIDevX AI combines innovation with usability, enhancing productivity across diverse tasks.

AIDevX AF

1. Managing Roles

1.1. Creating a New Role

  • Log in to the AIDevX platform using admin credentials.
    • Confirm you have permissions to manage roles.
    • For login issues, use the recovery feature or contact support.

Figure 1: Login Page

  • Navigate to Manage User > Role List.
    • View all roles, including their names, descriptions, and statuses.
    • Use the search function for quick navigation.

Figure 2: Navigate to Role List

  • Click Add Role to open the role creation form.

Figure 3: Add Role Form

  • Fill in the form fields:
    • Name: Provide a unique role name.
    • Description: Describe the role’s purpose and responsibilities.

Figure 4: Add Role Form

  • Assign users to the role:
    • Select users from the list or click Add User to create new accounts.
    • Use the All-Users option to assign the role universally.
  • Set the status:
    • Active: Enable the role immediately.
    • Inactive: Keep the role disabled until further action.
  • Click Save to complete the process.
    • A confirmation message will appear.

2. Managing Users

2.1. Adding a New User

  • Log in to the AIDevX platform as an administrator.
  • Go to Manage User > User List.
    • View all registered users, their roles, and statuses.
    • Use search or filter tools for efficient navigation.

Figure 5: User List Overview

  • Click Add User to open the user creation form.

Figure 6: Add User Form

  • Complete the form:
    • Name: Enter a unique username.
    • Full Name: Provide the user’s full legal name.
    • Email: Input a valid email address.
    • Phone: Add the contact number.
    • Roles: Assign roles from the available list or add new roles.
    • Status: Choose Active or Inactive to control account accessibility.
  • Click Save to create the account.
    • A confirmation message will confirm the action.

3. Managing Menus

3.1. Adding a New Menu

  • Log in to the AIDevX platform.
  • Navigate to Menu Editor.
    • Use the All Menu tab to view existing menus.
    • Search for specific menus easily.

Figure 7: All Menu Overview

  • Click Add Menu.

Figure 8: Manage Side Menu Overview

  • Fill in the form:
    • Title: Specify the menu title.
    • Path: Define the URL path (e.g., /path).
    • Parent Menu: Choose a parent menu if applicable.
    • Visibility: Control user access with this toggle.
  • Click Save to finalize.
    • The menu will appear in the All Menu list.

3.2. Managing Side Menus

  • Go to Manage Side Menu.

Figure 8: Manage Side Menu Overview

  • Drag items from All Menu to the Side Menu list.
    • Group menus logically by adding headers.
  • Click Save to apply changes.

4. Chat Tab

4.1. Uploading Document / Image

  • Locate the upload option in the chat interface.
  • Select a file to upload.
  • Confirm the upload.

4.2. Using Saved Prompt

  • Open the prompt menu.
  • Select a saved prompt.
  • The prompt will be attached in the chat.

4.3. Copy Reply

  • After receiving a reply, click the Copy button.
  • Paste it wherever needed.

4.4. Send to Editor

  • Use the Send to Editor button to transfer a response.
  • Edit and format in the editor as needed.

4.5. Regenerate Reply

  • Click the Regenerate button for a new response.
  • Review the updated reply.
